CRIAÇÃO DE SITES

Html5

É uma linguagem de marcação para a World Wide Web, utilizada para a construção de páginas.

CSS3

É um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

JavaScript

É atualmente a principal linguagem para programação client-side, e também é bastante utilizada do lado do servidor com o node.js.

Bootstrap

O framework front-end e de código-aberto, criado para o desenvolvimento web mais rápido e prático.

PHP

É uma linguagem interpretada livre, usada para o desenvolvimento de aplicações no lado do servidor, capazes de gerar conteúdo dinâmico na Web.

MySql

O MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface.

WordPress

É um sistema livre e aberto de gestão de conteúdo para internet, voltado principalmente para a criação de (sites) e blogs.

Faq

DÚVIDAS FREQUENTES

Como é o processo construção de um Site?

Varia de site para site, porém os mais comuns são construidos utilizando CMS como Wordpress.

Quanto custa a criação de um site?

Depende do tipo de site, alguns sites precisam de sistemas mais avançados, logo são mais caros, outros sistemas mais simples variam com preço mais barato.

Tenho que pagar mensalidade?

Sim, um site requer uma hospedagem que normalmente cobra um valor fixo todo mês para manter os arquivos do site no ar.

O que é um domínio?

Um domínio basicamente é o nome do site por exemplo, www.google.com.br é o nome do domínio do Google.
